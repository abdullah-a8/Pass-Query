# pp - Proton Pass Search

Fast password search for Proton Pass CLI.

## Usage

```bash
pp <search-term>
```

Searches all vaults in parallel and prints the password to stdout.

## Requirements

- [pass-cli](https://protonpass.github.io/pass-cli/) installed and logged in
- Rust 1.70+

## Installation

```bash
cargo install --path .
```
